# Estruturas de Dados

Este projeto reúne a resolução de uma atividade prática desenvolvida em JavaScript utilizando conceitos essenciais de estruturação de dados: **Vetores (Arrays)**. Essa estrutura é uma das bases fundamentais da linguagem, permitindo agrupar, organizar e manipular informações de forma lógica e eficiente.

A aplicação dessas estruturas confere organização aos algoritmos, simulando como dados reais são gerenciados na memória antes de serem processados ou exibidos.

---

### Atividades Desenvolvidas

* **#1 - Inversão de nomes**
Faça um programa que leia 7 nomes de pessoas e guarde-os em um vetor. No final, mostre uma listagem com todos os nomes informados, na ordem inversa daquela em que eles foram informados.

* **#2 - Média e valores acima da média**
Faça um programa que leia um vetor com 8 números reais e calcule a média dos valores armazenados. Após isso, exiba os valores do vetor que estão acima da média.

* **#3 - Filtro de menores de idade**
Crie um programa que leia o nome e a idade de 9 pessoas e guarde esses valores em dois vetores, em posições relacionadas. No final, mostre uma listagem contendo apenas os dados das pessoas menores de idade (nome e idade).

* **#4 - Leitura de números comuns em vetores**
Leia dois vetores de 5 elementos cada (valores inteiros). Crie um terceiro vetor que contenha apenas os elementos que estão em ambos os vetores (interseção, sem repetição). 
Exemplo: 
Vetor 1 → [1, 2, 3, 4, 5] 
Vetor 2 → [3, 4, 5, 6, 7] 
Resultado → [3, 4, 5]

---

### Considerações Finais

Esta atividade é um passo essencial para compreender como o JavaScript estrutura informações no mundo real. Dominar a manipulação de vetores nos prepara para lidar com fluxos de dados mais complexos e dinâmicos em projetos futuros.
